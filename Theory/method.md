# 单模双频贴片天线尺寸计算方法

> 基于论文: *Single-Mode Dual-Band Patch Antenna Based on the Lorentz Model of Dispersive Metamaterials*
> J. Yin et al., IEEE Trans. Antennas Propag., vol. 71, no. 11, Nov. 2023.

---

## 一、核心原理

### 1.1 传统贴片天线尺寸公式

矩形贴片天线在 **TM10 主模** 下工作时，其谐振长度由以下公式决定（论文公式 10）：

```
L = c0 / (2 * f * sqrt(er * ur)) = c0 / (2 * f * n)
```

其中：
- c0 = 3e8 m/s — 自由空间光速
- f — 工作频率 (Hz)
- er — 基板相对介电常数
- ur — 基板相对磁导率
- n = sqrt(er * ur) — 基板折射率

**关键含义**：贴片长度 L 由 **频率 f** 和 **折射率 n** 共同决定。对于常规介质基板（n 为常数），一个贴片只能在一个频率谐振。

### 1.2 Lorentz 色散材料的独特性质

Lorentz 模型的复介电常数（公式 1）：

```
er_tilde(w) = 1 + wp^2 / (w0^2 - w^2 - j * w * g0)
```

其中：
- wp = 2*pi*fp — 等离子体角频率
- w0 = 2*pi*f0 — 谐振角频率
- g0 — 阻尼系数

**关键特性**（如图 1 所示）：
- **低于谐振频率** (f < f0)：Re(er) 较大 → **高折射率**
- **高于谐振频率** (f > f0)：Re(er) 较小 → **低折射率**

### 1.3 单模双频条件

如果基板折射率 n 随频率变化，使得（公式 9）：

```
f1 * n(f1) = f2 * n(f2)
```

则两个频率在介质中的 **波长相同**：

```
lambda(f1) = c0 / (f1 * n(f1)) = c0 / (f2 * n(f2)) = lambda(f2)
```

因此 **同一个物理尺寸 L 的贴片** 可以在两个频率下都以 TM10 模式工作。

## 二、设计流程

### 步骤 1：确定目标频率

选择两个工作频率 f1（低频）和 f2（高频），以及 SRR 谐振频率 f0，需满足：

```
f1 < f0 < f2
```

论文示例：
- f1 = 2.6 GHz
- f2 = 4.1 GHz
- f0 = 3.1 GHz

### 步骤 2：设计 SRR 单元实现 Lorentz 色散磁导率

SRR（开口谐振环）的等效磁导率遵循 Lorentz 模型（公式 11）：

```
ur_tilde(w) = 1 + wp^2 / (w0^2 - w^2 - j * w * g0)
```

SRR 的几何参数决定 f0、wp、g0：
- **环长 l3**：主要控制谐振频率 f0
- **环间距 l2**：影响谐振频率
- **周期 p**：影响 f2/f1 比值

论文中 SRR 单元尺寸（图 10）：
- l1 = 2.8 mm, l2 = 1.75 mm, l3 = 8.0 mm
- h1 = 3.0 mm, h2 = 1.0 mm
- p = 12.5 mm（单元周期）
- 单元大小约 0.12*lambda0 x 0.12*lambda0 x 0.05*lambda0

### 步骤 3：计算两个频率下的有效折射率

SRR 提供色散磁导率 ur_tilde(f)，基板提供介电常数 er（论文使用 F4BM220，er = 2.2）。

有效折射率：

```
n(f) = Re[ sqrt(er * ur_tilde(f)) ]
```

在 f1 处：ur(f1) > 1 → n(f1) 较大
在 f2 处：ur(f2) < 1 → n(f2) 较小

### 步骤 4：验证双频条件

检查是否满足：

```
f1 * n(f1) = f2 * n(f2)
```

若不满足，调整 SRR 参数（l3、p 等）重新计算。

### 步骤 5：计算贴片尺寸

使用公式 10 计算贴片长度：

```
L = c0 / (2 * f1 * sqrt(er * ur(f1)))
  = c0 / (2 * f2 * sqrt(er * ur(f2)))
```

贴片宽度经验公式：

```
W = (c0 / (2 * f1)) * sqrt(2 / (er + 1))
```

### 步骤 6：CST 仿真优化

由于实际 SRR 数量有限（论文使用 2x4 阵列），有效参数与无限周期边界条件不同，需通过全波仿真优化以下参数：

| 参数 | 说明 |
|------|------|
| L2, W2 | 贴片长度和宽度 |
| W3 | 耦合槽长度 |
| L4, W4 | 馈线长度和宽度 |
| p | SRR 单元周期 |
| l1, l2, l3 | SRR 几何尺寸 |

## 三、论文天线结构详解

### 3.1 整体结构（图 13）

天线由 **4 层基板** 堆叠而成：

```
Layer 1 (H1=1.0mm): 贴片 (Patch)
Layer 2 (H2=3.0mm): SRR 超材料 (双金属层 + 金属化过孔)
Layer 3 (H3=1.0mm): 纯介质间隔层
Layer 4 (H4=1.0mm): 地层 + 微带馈线
```

### 3.2 馈电方式

采用 **孔径耦合馈电 (aperture-coupled feed)**：
- 底层微带线通过地层上的槽缝耦合能量到贴片
- 避免馈电结构干扰 SRR 层

### 3.3 最终尺寸

| 参数 | 值 | 说明 |
|------|-----|------|
| L2 | **37.5 mm** | 贴片长度 |
| W2 | **25.5 mm** | 贴片宽度 |
| L1, W1 | 100 mm | 整体基板尺寸 |
| W3 | 120 mm | 耦合槽长度 |
| W4 | 3.0 mm | 馈线宽度 |
| D1 | 1.0 mm | 过孔直径 |
| G | 0.2 mm | SRR 开口间隙 |

## 四、计算示例

### 4.1 参数设定

以论文设计为例：

| 参数 | 值 |
|------|-----|
| f1 | 2.6 GHz |
| f2 | 4.1 GHz |
| f0 | 3.1 GHz |
| fp | ~5.0 GHz |
| g0 | 0.2 |
| er | 2.2 |

### 4.2 计算过程

**1) 计算 f1 处的有效磁导率：**

w1 = 2 * pi * 2.6e9 = 1.634e10 rad/s

w0 = 2 * pi * 3.1e9 = 1.948e10 rad/s

wp = 2 * pi * 5.0e9 = 3.142e10 rad/s

ur_tilde(f1) = 1 + (3.142e10)^2 / ((1.948e10)^2 - (1.634e10)^2 - j * (1.634e10) * 0.2)

**2) 计算 f2 处的有效磁导率：**

w2 = 2 * pi * 4.1e9 = 2.576e10 rad/s

ur_tilde(f2) = 1 + (3.142e10)^2 / ((1.948e10)^2 - (2.576e10)^2 - j * (2.576e10) * 0.2)

**3) 计算有效折射率：**

n(f1) = Re[ sqrt(2.2 * ur_tilde(f1)) ]

n(f2) = Re[ sqrt(2.2 * ur_tilde(f2)) ]

**4) 验证条件：**

f1 * n(f1) = f2 * n(f2) ?

**5) 计算贴片长度：**

L = 3e8 / (2 * 2.6e9 * n(f1))  (单位: m)

### 4.3 预期结果

- n(f1) > n(f2)（因为 f1 < f0 < f2）
- f1 * n(f1) = f2 * n(f2)
- L = 37.5 mm（与论文一致）

## 五、注意事项

### 5.1 损耗控制

- 工作频率应避开 SRR 谐振频率附近的高损耗区域（图 3 灰色区域）
- 论文中 f1 处效率 98%，f2 处效率 95%

### 5.2 SRR 各向异性

SRR 仅对 **垂直于环面的磁场** 响应（图 8a）：
- SRR 平面应平行于贴片，使 TM10 模式的 H 场穿过 SRR 环面
- 馈电网络的 H 场不与 SRR 耦合，因此馈电引入的伪谐振不会映射到高频段

### 5.3 制造公差

- 层间空气隙会使工作频率升高（论文中 0.15 mm 空气隙导致频率偏移）
- PCB 制造公差也会影响实测频率

### 5.4 扩展性

- 若天线工作在 N 个模式，理论上可获得 2N 个工作频段
- 该方法同样适用于色散介电常数超材料
